# 🕉️ Sanskrit Vākya Synthesizer

A rule-based Sanskrit chanting engine that converts text into rhythmic speech using phonetic modeling.

## 🚀 Features

- 🔤 English → Sanskrit conversion
- 🕉️ Chant-like speech generation
- 🎚️ Adjustable chanting speed
- 🧠 Phonetic elongation & rhythm engine
- 💻 Fully offline (no API dependency)

## 🧪 Example

Input:
om namah shivaya

Output:
ॐ नमः शिवाय (chant-style)

## 🛠️ Tech Stack

- React + TypeScript
- Web Speech API
- Node.js (Express)

## 🎯 Use Case

Designed for:
- Sanskrit learning
- Chanting simulation
- Educational tools

## 🏆 Innovation

Instead of using external APIs, this project:
- Uses phonetic transformations
- Applies rhythm-based timing
- Simulates chanting patterns
